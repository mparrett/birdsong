# Experiments

This directory holds research branches that are intentionally separate from the
supported root modem in `birdsong.py`.

- `active/` contains the small set of experiments still worth exercising.
- `archive/` contains preserved branches that are superseded, incomplete, or
  mainly useful for historical comparison.

Promotion rule:

- An experiment should only live in `active/` if it still represents a distinct
  idea and can pass at least a basic loopback smoke test.
