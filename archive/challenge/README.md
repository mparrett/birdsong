# Challenge Archive

This directory preserves the coursework/challenge branch that fed into the main
Birdsong repo. It is historical material, not a supported surface.

Contents:

- `coursework_solution_modem.py`: the archived single-file modem solution built
  for the coursework constraints.
- `dependency_notes.md`: preserved notes from an early dependency discussion.
- `course_material/`: the original assignment prompt, agent guidance, and starter
  scaffold kept for reference.

Guidelines:

- Do not treat this directory as current product code.
- Prefer reading it for historical context, constraints, or ideas that may still
  be useful.
- If any archived artifact becomes relevant again, promote it intentionally
  rather than referencing it as if it were still active.
