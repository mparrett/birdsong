# History Archive

This directory holds early prototypes and other preserved historical code that
is no longer part of the supported or actively researched surface.

Contents:

- `poc_bootstrap_modem.py`: the original file-based proof of concept used before
  the current modem shape settled.

Use this area for reference or comparison only. If a historical prototype is
worth reviving, it should be promoted deliberately rather than edited in place as
if it were current code.
