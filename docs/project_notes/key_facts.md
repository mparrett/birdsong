# Key Facts

## Audio Parameters
- Sample rate: 44.1 kHz
- FSK frequencies: 196 Hz (G3) / 1760 Hz (A6)
- Bit duration: 50ms (20 bits/s)
- Handshake tone: C8 (4186 Hz)

## Supported Surface
- Supported entrypoint: `birdsong.py`
- Root validation commands: `just check`, `just test`, `just e2e`,
  `just e2e-pipes`
- Root philosophy: keep the main modem small and keep experiments/history out of
  the supported path
